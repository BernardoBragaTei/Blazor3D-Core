Object3D

NameSpace: Blazor3D.Core

It's a base abstract class for most objects in Blazor3D. It provides functionality for manipulating objects in 3D space.

Wrapper for three.js Object3D

Methods

Add(Object3D)
Adds a child Object3D object to the Children collection.

Parameters

child
Child Object3D to be added.

Properties

Position
Represents the object's local Vector3 position. Default is (0, 0, 0).
Rotation
Object's local Euler rotation, in radians.
Scale
Represents the object's local Vector3 scale. Default is (1, 1, 1).
Name
Optional name of the object. Default is an empty string. It has not to be unique.
Uuid
Universal unique identifier of this object instance. It's automatically assigned Guid, so it shouldn't be edited.
Children
Collection of Object3D child objects.
